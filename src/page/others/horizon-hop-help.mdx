## 项目简介

**Horizon-Hop** 是一款轻量、克制、但非常实用的浏览器效率插件。

我们希望把那些你在浏览器里会高频用到的小能力，认真做成一个顺手、好看、打开就能用的工具集合。它不是一个什么都想做的大而全平台，而是一套更聚焦的浏览器工作台，帮你把二维码生成、JSON 处理、图片压缩、变量命名、表格结构整理这些常见操作，留在当前浏览器里完成。

如果你经常在网页、文档、接口数据、临时素材和开发辅助工具之间来回切换，Horizon-Hop 会是一个刚刚好的补充层。

## 现在的 Horizon-Hop 能做什么

目前插件主要围绕以下几类场景展开：

- **当前网页二维码生成**：打开首页即可为当前网页生成二维码，支持自定义内容与下载。
- **二维码工坊**：进一步生成更精致的二维码和分享海报，支持历史记录、多方案保存与回看。
- **JSON 工具箱**：支持格式化、校验、结构预览、TypeScript 类型生成和代码输出。
- **图片压缩**：本地压缩图片、批量处理、预览对比、格式转换，兼顾效率和隐私。
- **变量命名助手**：输入中文语义，快速得到更像样的变量名建议。
- **表格结构整理**：把文档表格内容整理为更可读的 JSON 结构，方便接口文档和字段梳理。
- **书签搜索与功能面板**：通过快捷键快速唤起常用能力，减少鼠标路径。
- **每日一句侧边栏**：在侧边栏保留一个更轻、更安静的落点。
- **版本更新日志**：每次升级后可直接查看更新内容，方便了解新能力和变化。

## 如何开始使用

安装并启用 Horizon-Hop 后，你可以通过下面几种方式进入主要能力：

### 1. 点击扩展图标

点击浏览器工具栏中的 Horizon-Hop 图标，即可打开首页弹窗。

首页会展示当前网页二维码，同时提供常用快捷入口，适合处理最常见、最轻量的操作。

### 2. 使用快捷键

如果你更偏爱键盘流，可以直接使用这些快捷键：

| 快捷键  | 功能         |
| ------- | ------------ |
| Alt + O | 打开首页弹窗 |
| Alt + P | 打开功能面板 |
| Alt + B | 打开书签搜索 |
| Alt + S | 打开侧边栏   |

### 3. 从工具页直接工作

对于 JSON、二维码、图片压缩、变量命名、表格结构整理这类更完整的能力，插件会以独立工具页的形式打开，适合持续处理内容，而不是在小弹窗里挤着操作。

## 首页弹窗怎么用

当前版本的首页弹窗已经收敛成一个更清晰的入口页，重点放在两件事上：

- 快速为当前网页生成二维码
- 快速跳转到真正需要展开使用的工具页

在首页中，你可以：

- 查看当前网页二维码
- 修改二维码内容，并实时重新生成
- 下载二维码图片
- 一键恢复为当前网页地址
- 进入二维码工坊查看更多高级能力
- 通过快捷入口唤起功能面板、书签搜索、侧边栏和更新日志

如果你只是临时分享网页、保存链接、发给手机扫码打开，首页已经足够快。

## 核心功能说明

### 一. 二维码工坊

二维码工坊是这一轮重构后的核心能力页之一。

相比首页里的轻量二维码，它提供了更完整的生成与管理体验。

**你可以在这里完成：**

- 为当前网页或任意文本生成二维码
- 调整尺寸、边距、颜色和容错级别
- 生成适合分享的二维码海报
- 自定义海报标题和描述
- 保存多个二维码方案
- 在历史记录中快速切换、回看和删除旧方案
- 下载二维码原图或海报图

如果你经常需要把网页、文档、活动链接、内部页面或临时内容转成二维码，这一页会比传统在线二维码工具更顺手。

### 二. JSON 工具箱

JSON 工具箱适合处理接口返回、配置对象、示例数据和临时结构整理。

**主要能力包括：**

- 自动格式化 JSON / 类 JSON 内容
- 错误校验和结构预览
- 视图模式切换
- 生成 TypeScript `type`
- 生成 TypeScript `interface`
- 生成 JS 可用结构输出
- 调整折叠、缩进、键名展示等查看配置
- 快速复制输出内容

如果你平时会在接口联调、后端返回排查、字段建模或文档整理时频繁接触 JSON，这一页可以明显减少手动整理的时间。

### 三. 图片压缩

图片压缩页专注于本地处理，尽量让文件压缩这件事更直觉。

**主要能力包括：**

- 拖拽或批量导入图片
- 本地压缩，不经过第三方服务器
- 调整压缩质量
- 查看压缩前后预览和体积对比
- 单张下载或批量下载
- 转换图片格式
- 清空和管理当前文件列表

适合在发图、提工单、上传后台、写文档或提测前快速把素材整理好。

### 四. 变量命名助手

变量命名助手适合在写代码、起字段名、组织组件状态或整理函数语义时使用。

**它可以帮助你：**

- 输入中文描述，快速生成变量名建议
- 输出更像开发场景里会真正使用的命名
- 减少“这个变量到底该怎么起名”的犹豫时间

这个功能的重点不是炫技，而是让你在思路不断的时候少停一下。

### 五. 表格结构整理

表格结构整理页适合处理接口文档、参数表格和字段说明表。

**你可以用它来：**

- 把表格内容整理成 JSON 结构
- 辅助理解嵌套对象和数组关系
- 快速得到更适合继续加工的数据结果

如果你经常从语雀、文档平台或内部说明页里复制字段表，这一页会很实用。

### 六. 书签搜索

书签搜索适合处理“我记得收藏过，但一时找不到”的场景。

**主要体验包括：**

- 快速唤起搜索面板
- 关键词模糊匹配书签
- 点击结果直接打开目标网址
- 适合键盘优先的使用方式

对于收藏很多资料、经常回看旧页面的人来说，这个入口会比手动打开书签栏更快。

### 七. 侧边栏与每日一句

侧边栏目前刻意收得很轻，只保留每日一句。

我们希望它先成为一个安静的落点，而不是急着塞进更多功能。后续如果有真正值得留下来的侧边栏能力，会继续在这里扩展。

### 八. 更新日志

插件升级后，如果当前版本存在对应的更新日志，Horizon-Hop 会自动打开更新日志页。

你可以在这里查看：

- 当前版本的主要变化
- 历史版本记录
- 每一版的重点功能调整

这部分后续也会持续沉淀，方便长期维护和用户感知版本变化。

## 适合哪些人使用

Horizon-Hop 适合这些场景：

- 经常处理网页链接、二维码和分享内容的人
- 经常联调接口、查看 JSON、写前端代码的人
- 经常整理图片素材、压缩文件、处理格式的人
- 希望把高频小工具尽量留在浏览器里的人
- 喜欢轻量工具，而不是重平台的人

## 使用建议

如果你第一次开始使用，我建议这样试：

1. 先打开首页弹窗，体验当前网页二维码
2. 再进入二维码工坊，看看高级生成和历史记录
3. 打开 JSON 工具箱，体验格式化和类型生成
4. 最后试一下图片压缩或书签搜索，找到最适合你的那几个高频入口

Horizon-Hop 最适合的使用方式，不是一次把所有功能都记住，而是先找到两三个你会反复用到的入口，让它慢慢变成浏览器里的默认动作。
